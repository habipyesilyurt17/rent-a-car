## Getting Started

To get this project copy up and running follow these simple steps.

### Prerequisites

- [Node.js](https://nodejs.org/en/)

### Installation

1. Clone the repo
   ```sh
   git@github.com:habipyesilyurt17/rent-a-car.git
   ```
2. Install packages
   ```sh
    npm install
    # or
    yarn install
   ```
3. Run the development server
   ```bash
   npm run dev
   # or
   yarn dev
   ```
